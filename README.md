📊 GetShop Today – E-Commerce Marketing Analytics Dashboard
🔍 Project Overview
This project analyzes marketing campaign performance for GetShop Today, focusing on spend efficiency, ROI, conversions, CPA, CPC, and revenue generation across multiple campaign types.

🎯 Objectives 
Evaluate campaign spend efficiency

Analyze ROI and revenue contribution

Compare CPA and CPC across channels

Identify top-performing campaign types

Support data-driven marketing decisions

📈 Key Metrics Analyzed
Total Spend

ROI

Total Clicks

CPA

Conversion Volume

Revenue by Campaign

CPC & CTR by Channel

🛠 Tools Used
Microsoft Excel

Pivot Tables

Data Cleaning Functions

Dashboard Design & Data Visualization

📊 Dashboard Preview

💡 Key Insights
Influencer campaigns generated the highest conversions.

PPC produced the highest revenue contribution.

CPA varied significantly across campaign types.

Email and Affiliate campaigns showed strong ROI efficiency.

📂 Dataset
The dataset includes campaign spend, clicks, conversions, revenue, and marketing channels.
